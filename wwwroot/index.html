md2html

  • md2html is a flat-file PHP framework that dynamically serves GitHub-flavored Markdown pages
  • This repo is running md2html on an Apache server inside Docker
    • The md2html back-end lives in /var/www/md2html (outside the web folder) and is designed to be updated using git
    • md2html/resources is symbolically linked to www/resources, and web-accessible files (CSS/JS) are placed here
    • Markdown content lives in sub-folders of /var/www/html and could be sync'd with separate git repos (one per project)
  • To create a new site make a new folder and into it copy:
    • md2html/page.html - page template
    • md2html/article.html - article template
    • md2html/serveIndex.php - php script that renders the pages
    • .htaccess - rule to point requests to the php script
